## Chromeへのインストール

1. [https://github.com/riiimparm/md-looks-good/releases](https://github.com/riiimparm/md-looks-good/releases) からzipダウンロード
2. [chrome://extensions](chrome://extensions)にて「デベロッパーモード」を有効化
3. 「パッケージ化されていない拡張機能を読み込む」
4. ディレクトリを選択
